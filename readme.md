# Invoice Generator (Single-File App)

A super simple invoice generator built as a single HTML file with zero dependencies and no build process.

Runs entirely in the browser with automatic data persistence via `localStorage`.

---

## Features

- Create invoices directly in the browser
- Fully offline-first (no backend required)
- Zero dependencies — pure HTML, CSS, and JavaScript
- Auto-save to `localStorage`
- PDF export via browser print functionality
- Currency support: USD, EUR (easily extendable)
